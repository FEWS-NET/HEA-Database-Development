{% translate 'An HEA Baseline for a LivelihoodZone in a given reference year' %}
{% blocktrans %}

Each LivelihoodZoneVersion contains the uploaded BSS that was used to create it.

The reference year is a consumption year, the beginning and the end of the year depends on the livelihood zone. 

  For non-Urban Livelihood Zones it is aligned with the Seasons.
  
  For agricultural and agropastoral Livelihood Zones,
  the reference year begins with the first month of the
  harvest and runs until the end of the lean season.
  For example, this is from October 2022 to September
  2023 for the Sahel countries.
  
  For pastoralist Livelihood Zones, the reference
  year begins with the month in which the livestock
  have enough fodder (providing good milk yield) and
  runs until the end of lean season for livestock.
  Example from July 2022 to June 2023 for
  the Sahel countries.
{% endblocktrans %}
